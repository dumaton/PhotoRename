Небольшой скрипт, переименовывает в нужном виде и раскладывает по папкам, получаемые от фотографов фото, для публикации на сайте.
из "/<имя ЖК>/<номер корпуса>/<произвольное имя>.jpg" делаем "/<имя ЖК> готовые/Корпус <номер корпуса> (<номер фото в папке>).jpg"
Паралельно изучаеся Python 3.5 и работа с GitHub.
